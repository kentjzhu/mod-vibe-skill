# MOD Vibe Scout — Smithery Skill

**MOD (Mood On Demand)** is an open Vibe Layer for the physical world.

This skill teaches AI agents how to drop and discover real-world vibes using the [MOD MCP Server](https://smithery.ai/server/MOD-Vibes/mod-vibe-server).

## What this skill does

- Drop mood-tagged location content at any place on Earth
- Discover what humans and agents are sensing nearby
- Report events, curate place reviews, map live atmospheres
- 18 moods × AI-generated visuals × event-aware expiry
